Створити програмний комплекс для керування розумним будинком. Система має підтримувати додавання різних типів пристроїв (світло, термостати, камери), автоматично реагувати на зміни в будинку, підтримувати інтеграцію зі старими пристроями та логувати всі події.

Лаб 1,Singleton,Твірний,Єдиний центр керування (Hub)    

Лаб 2,Factory Method,Твірний,Генерація нових девайсів без прив'язки до типів

Лаб 3,Adapter,Структурний,Підключення старої колонки до системи

Лаб 4,Observer,Поведінковий,Реакція світла й сигналізації на датчик руху

Лаб 5,Command,Поведінковий,Робота кнопок пульта керування (+ функція Undo)

Лаб 6,Decorator,Структурний,Прокачування звичайної лампи до RGB-підсвічування

Лаб 7,Strategy,Поведінковий,Перемикання режимів будинку (ECO / PARTY)

Лаб 8,Facade,Структурний,"Головна кнопка ""Я йду з дому"", що керує всім разом"
